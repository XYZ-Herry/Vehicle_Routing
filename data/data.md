## 环境及依赖

```cpp
环境：python3
依赖：
pandas：用于处理 Excel 文件和数据操作。
numpy：用于数值计算。
openpyxl：用于处理 Excel 文件和数据操作。
安装库：
pip install pandas numpy openpyxl
```

## 运行命令及输入参数

```cpp
python3 data.py
参数：
  -h, --help  show this help message and exit
  -vu VU      无人机速度 (默认 20)
  -vc VC      车辆速度 (默认 10)
  -cu CU      无人机运送货物的成本 (默认 0.84)
  -cc CC      货车运送货物的成本 (默认 0.62)
  -om OM      配送时间权重 (默认 0.1)
  -qu QU      无人机最大载重 (默认 20)
  -fu FU      无人机最大电量 (默认 2.0)
```

## 输入格式

```cpp
起始需求数量
额外需求数量
车辆共配中心数量
无人机共配中心数量
```

## 输出格式

```cpp
起始需求数量 额外需求数量 车辆配送中心数量 无人机配送中心数量//第一行
无人机速度 车辆速度 无人机运送货物的成本 货车运送货物的成本 无人机最大载重 无人机电量 配送时间权重 //第二行
边的数量//第三行
边的节点1 边的节点2 边的长度//边的数量行，会有重边但是重边的长度是一致的
需求点 x坐标 y坐标 取货重量 送货重量//默认需求点数量行
车辆配送中心（起始位置）序号 车辆配送中心（起始位置）x坐标 车辆配送中心（起始位置） y坐标 车辆配送中心（起始位置）车辆数// 车辆数量行
无人机配送中心（起始坐标）序号 无人机配送中心（起始坐标）x坐标 无人机配送中心（起始坐标）y坐标 无人机配送中心（起始位置）无人机数// 无人机数量行
需求点 x坐标 y坐标 取货重量 送货重量 时间//额外需求点行
节点1 节点2 早高峰速度系数 晚高峰速度系数//任意两点之间的速度系数行
```

## 额外说明

```
1.暂时没有堵车信息因为还没有堵车的excel表格
2.x坐标y坐标为经纬度，无人机送货时走直线距离可用这个计算
3.车辆速度和无人机速度的单位为km/h
4.额外需求点的时间单位为min
5.边的长度单位为米
6.早高峰和晚高峰速度系数范围为0.2-1.0，表示车辆在该时段的速度降低比例
```
